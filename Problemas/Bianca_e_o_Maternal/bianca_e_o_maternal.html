Bianca e o Maternal

TL: 2 segundos
ML: 128 MB
Bianca é uma professora de maternal para crianças prodígias. Ela quer saber a nota que seus alunos dão para ela. Para isso, aproveitando a fila da cantina, Bianca pergunta para cada um dos seus alunos qual a nota que ele dá pra ela e resolve aceitar a maior nota como a nota verdadeira. Tudo seria muito simples se as crianças do maternal prodígio não adorassem política, democracia e discussão. Cada criança pergunta para seu melhor amigo a nota que ele daria para a Bianca e diz para a professora a média entre a nota que ela daria e a nota que seu amigo respondeu. O melhor amigo dessa criança realiza o mesmo processo. Se uma criança que já foi perguntada for perguntada novamente por outra criança, ela cansa do processo e não pergunta mais, dizendo apenas a nota que ela dá para Bianca. Quando o Bianca termina de perguntar para uma criança, todos se renovam e estão dispostos a realizar o processo novamente. Ajude Bianca a dizer qual é a sua melhor nota e para qual criança deverá perguntar para obter esse resultado.

Entrada

São diversos casos de teste. Cada caso de teste começa com um inteiro N (1 ≤ N ≤ 1000), o número de crianças na fila da cantina. Seguem N linhas, cada uma com dois inteiros, A (0 ≤ A ≤ 10), a nota que a i-ésima criança dá para Bianca, e B(0 ≤ B < N), o índice da criança que é melhor amiga da i-ésima criança. Uma criança pode ser melhor amiga dela mesma. Triste, né?

Saída

Para cada caso de teste, imprima uma linha com dois números separados por espaço. O primeiro número é a melhor nota que Bianca consegue, com 3 casas decimais de precisão, e o segundo é o índice da criança para qual deve ser realizada a pergunta para obter essa nota. Se mais de uma criança responder a nota máxima, imprima a de menor índice.

Exemplos

Entrada de Teste 3 5 1 10 0 8 1
Saída de Teste 8.750 1
