Roberterson Técnico de Futebol

TL: 2 segundos
ML: 128 MB
Roberterson é o técnico de um time de futebol com N jogadores. Como eles sempre ficam em último, neste ano eles resolveram participar de um campeonato com regras experimentais, para tentar a sorte. Neste campeonato, o número de jogadores que podem entrar em campo em cada jogo é diferente para cada partida. Apesar das derrotas, Roberterson conhece muito bem o seu time e sabe o quão bom um jogador é e qual é o efeito de sua presença na qualidade dos outros jogadores que estão em campo. Mais especificamente, ele possui uma planilha com uma lista de N inteiros indicando a habilidade de cada jogador e uma tabela com N linhas e N colunas contendo um inteiro Aij em cada linha i e coluna j indicando quanta habilidade deve ser somada ao jogador j quando o jogador i também está em campo. Sabendo que na próxima partida deverão entrar em campo M jogadores, imprima a máxima soma das habilidades dos jogadores se Roberterson escolher o melhor time.

Entrada

A entrada será composta de múltiplos casos de teste. Na primeira linha de cada caso serão dados dois inteiros N (1 ≤ N ≤ 20) e M (1 ≤ M ≤ N). Na segunda linha serão dados N inteiros indicando a habilidade Hi do jogador i (-100000 ≤ Hi ≤ 100000). Nas próximas N linhas serão dados N inteiros em cada linha i, sendo que o j-ésimo inteiro é o Aij na tabela de Roberterson (-100000 ≤ Aij ≤ 100000). É garantido que Aij = 0 para i = j.

Saída

Imprima a somatória das habilidades dos jogadores em campo se Roberterson escolher o melhor time.

Exemplos

Entrada de Teste 4 3 10 3 2 2 0 -6 -4 -3 -1 0 2 3 -2 3 0 1 -5 2 3 0
Saída de Teste 21
