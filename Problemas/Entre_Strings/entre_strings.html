Entre Strings

TL: 2 segundos
ML: 128 MB
Você até agora achou que os problemas estão ordenados por dificuldade, não? Só que não. Este é o mais fácil da prova. Dadas duas cadeias de caracteres imprima a concatenação das duas!!

Entrada

A entrada contém vários casos de teste. Cada caso tem duas strings separadas por espaço, contendo apenas letras do alfabeto. As strings terão tamanho máximo de 10^4. Você deve ler até EOF.

Saída

Para cada caso de teste imprima a concatenação das duas strings.

Exemplos

Entrada de Teste sim nao tchau pardal
Saída de Teste simnao tchaupardal
