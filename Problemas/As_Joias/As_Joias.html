As jóias de Roberterson

TL: 3 segundos
ML: 128MB
Roberterson é um menino muito popular entre as garotas de sua sala, mas essa popularidade não vem de graça, ele tenta agradar ao máximo suas amigas dando para elas presentes de vez enquando. Certa manhã Roberterson resolveu comprar N jóias para distribuir entre as N meninas mais bonitas da sala, cada jóia é única e cada menina só pode receber um unico presente. Após comprar as jóias ele ficou em dúvida sobre qual dar para qual menina para maximizar a felicidade total delas ao receberem os presentes. Roberterson também sabe ler mentes e portanto ja sabe de antemão a felicidade para cada garota i ao receber a jóia j. A felicidade total das meninas é dada pela soma das felicidades individuais das mesmas ao receberem uma jóia.

Entrada

A entrada é composta de varios casos de teste. A primeira linha de cada caso contém um inteiro N(1 <= N <= 10). As próximas N linhas conteram cada uma N inteiros, um numero X(1 <= X <= 100) na linha i, coluna j indica que a menina i terá felicidade X ao receber a jóia j

Saída

Para cada caso de teste imprima um numero representando a máxima felicidade total das garotas

Exemplos

Entrada de Teste 2 1 2 2 1
Saída de Teste 4
