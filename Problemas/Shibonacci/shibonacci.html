Shibonacci

TL: 2 segundo (~50 casos)
ML: 128 MB
Shi gosta de inventar sequências. Desta vez ele criou um grupo de sequências que são uma generalização da sequência de Fibonacci, ele chamou esse grupo de sequências de Shibonacci. Na sequência de Fibonacci o X-ésimo Fibonacci pode ser obtido pela fórmula F(X) = F(X-1) + F(X-2). As sequências de Shibonacci possuem cada uma um número inteiro K associado à ela, sendo que SK(X) pode ser obtido pela somatória de SK(X - i) para i = 1 até i = K. Ou seja, a sequência de Fibonacci é uma sequência de Shibonacci com K = 2 (S2(X) = F(X)). Os casos bases das sequências de Shibonacci são dados por SK(i) = 1 para i = 0 até i = K - 1. Shi é preguiçoso e pediu para você escrever um programa que dado K e dado X ele retorne SK(X).

Entrada

A entrada possui vários casos de teste. Cada caso de teste terá uma linha com dois inteiros K (1 ≤ K ≤ 100) e X (0 ≤ X ≤ 10000).

Saída

Para cada caso de teste imprima SK(X) mod 1000000007.

Exemplos

Entrada de Teste 3 10
Saída de Teste 193
