O sonho de Bob

TL: 4 segundos
ML: 128 MB
Bob é um sapo com um sonho. No pantano em que ele vive, existe uma estranha escada de pedra que nenhum sapo jamais conseguiu subir. Essa escada é diferente das escadas normais pois a diferença entre as alturas de dois degraus consecutivos não são necessariamente iguais em toda a escada. O sonho de Bob é subir essa escada em até K pulos, e agora que os sapos cientistas conseguiram descobrir a altura de cada degrau usando matemágica, ele está determinado a treinar o suficiente para conseguir realizar o seu sonho. O problema é que os pulos de Bob não são altos o suficiente e ele gostaria de saber qual é a altura mínima que seus pulos precisam ter para que ele consiga subir a escada em no máximo K pulos. Note que se um pulo for suficiente para avançar mais de um degrau, ele pode avançar mais de um degrau por pulo.

Entrada

A entrada será composta de múltiplos casoso de teste. Na primeira linha de cada caso serão dados dois inteiros N e K (1 ≤ N ≤ 104) e (1 ≤ K ≤ N) indicando respectivamente o número de degraus na escada e o número máximo de pulos que Bob sonha dar para subir toda a escada. Na próxima linha, para i = 1 até i = N, serão dados inteiros Ai (1 ≤ Ai ≤ 107), indicando cada um a altura do i-ésimo degrau. É garantido que Ai ≤ Aj para todo i < j.

Saída

Para cada caso de teste imprima em uma linha um inteiro com a altura mínima que os pulos de Bob precisam ter para que ele consiga realizar o seu sonho.

Exemplos

Entrada de Teste 8 3 1 1 4 7 8 9 13 14
Saída de Teste 5
