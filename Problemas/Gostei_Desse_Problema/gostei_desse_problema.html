Gostei Desse Problema

TL: 2 segundos
ML: 128 MB
Shi é uma criança excêntrica. Em vez de brincar com seus amiguinhos na rua, ele prefere olhar o Gugou Earth e procurar palavras no dicionário. Shi propôs a você um desafio. Dadas as suas palavras preferidas, descubra se elas existem ou não no dicionário.

Entrada

O problema tem um único caso de teste. A primeira linha contém um número N (1 <= N <= 10^5), que indica quantas palavras existem no dicionário. As N linhas seguintes contêm as palavras listadas uma a uma, de tamanho máximo 100. Depois é dado um número Q (1 <= Q <= 10^5), que indica quantas strings devem ser procuradas no dicionário. As Q linhas seguintes contêm as palavras a serem procuradas, de tamanho máximo 100. Todas as palavras contém apenas letras do alfabeto.

Saída

Para cada palavra a ser procurada imprima "S" se ela existe ou "N" caso contrário.

Exemplos

Entrada de Teste 13 usem hahaha map waw de wat strings asa senao BoLa vai dar TLE 6 asa bola BoLa DE dE vaia
Saída de Teste S N S N N N
