A + B = C

TL: 1 segundo
ML: 128 MB
Esse problema é tão fácil que dispensa história: Dada uma sequência de N inteiros, existe algum par A+B dessa sequência que é igual a C?

Entrada

A entrada é composta de vários casos de teste. A primeira linha conterá um número N (3 <= N <= 10^5), o tamanho da sequência. A próxima linha conterá N inteiros, que são os números. Por fim, a última linha de cada caso, conterá um inteiro C. Os números vão de 0 até 10^9.

Saída

Imprima para cada caso de teste "S" se existe algum par A+B que seja igual a C, caso contrario imprima "N".

Exemplos

Entrada de Teste 3 1 2 4 4 2 2 3 5
Saída de Teste N S
