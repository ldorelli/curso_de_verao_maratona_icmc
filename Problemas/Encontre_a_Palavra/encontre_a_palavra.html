Encontre a Palavra

TL: 1s
ML: 128Mb
Algumas pessoas com tempo para bobagem se perguntam qual a maior palavra da língua portuguesa. Dada uma lista de palavras, encontre a maior palavra.

Entrada

Cada linha conterá uma palavra de tamanho máximo 1000. A palavra é formada apenas por números ou letras do alfabeto (maiúsculas também).

Saída

Imprima apenas uma linha contendo a maior palavra encontrada, em caso de empate imprima a que aparecer primeiro na entrada.

Exemplos

Entrada de Teste ola um testando123
Saída de Teste testando123
