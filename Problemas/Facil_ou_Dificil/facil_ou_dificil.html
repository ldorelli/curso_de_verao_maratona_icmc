Fácil ou Difícil?

TL: 2 segundos
ML: 128 MB
Uma string (cadeia de caracteres) formada por parênteses () é balanceada se ocorre o correto casamento dos parênteses. Por exemplo, “(((())))” e "(()())()" são balanceadas enquanto “))())” e "(" não são. Escreva um programa que leia uma string deste tipo e decida se a ela é ou não balanceada.

Entrada

Cada linha conterá uma string formada apenas por parênteses de tamanho máximo 10^4 e no mínimo 1. Você deve ler até EOF.

Saída

Para cada caso, imprima "S" se for balanceada ou "N" caso contrário.

Exemplos

Entrada de Teste ( )( () ()((())) (((()))) ))()) ()()()()() (((()()()()))) ((((((((((((( (((((((()))))
Saída de Teste N N S S S N S S N N
