A + B + C + D

TL: 1 segundo
ML: 128 MB
Denis tem uma lista de números preferidos. Porém, Denis não é imparcial sobre seus números e gosta mais de uns do que de outros. Por isso, os números preferidos de Denis aparecem mais de uma vez na lista. Shi roubou a lista de Denis e a dividiu em 4. Shi disse que só devolveria a lista caso Denis descobrisse se é possível obter um número E somando um número de cada lista. Como babá de Denis, você deve resolver este problema antes que Denis comece a chorar.

Entrada

Cada caso de teste possui 5 linhas. A primeira linha consiste no inteiro -10^5 <= E <= 10^5. As próximas 4 linhas começam com um número 1 <= Ni <= 25 que indica a quantidade de números da lista i, seguido de Ni números -10^5 <= aij <= 10^5 que compõem a lista i. Leia até EOF.

Saída

Para cada caso de teste, imprima "S" se for possível obter E e "N" caso contrário.

Exemplos

Entrada de Teste 0 3 1 2 3 3 1 3 5 2 -1 0 1 -3 -100 3 1 2 3 3 1 3 5 2 -1 0 1 -3
Saída de Teste S N
